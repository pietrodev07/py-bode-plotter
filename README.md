# py-bode-plotter

This repo contains all files of the basic implementation of the bode plotter in python.

## Run Locally

**Clone the respository**

```bash
  git clone https://github.com/pietrodev07/py-bode-plotter
```

**Go to the project directory**

```bash
  cd py-bode-plotter
```

**Start the program**

```bash
  python main.py
```
